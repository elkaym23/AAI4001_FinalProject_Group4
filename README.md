# Crop Yield Prediction Project

## Overview

This project aims to improve crop yield prediction using data-driven methods. Crop yields are influenced by many factors, including climate conditions, farming inputs, and management practices, which makes them difficult to predict accurately. Unreliable predictions can lead to poor planning, financial losses, and food security risks.

To address this challenge, the project uses a historical global agriculture dataset and machine learning techniques to better understand what drives crop yields and how they change under different conditions.

## Guide

### How to Run the FastAPI Backend

1. Open Terminal and type the below:

```bash
- Navigate to the backend folder: cd fastapi_backend
- pip install -r requirements.txt
- Start the API: uvicorn AAI4001_FinalProject_Group4_fastapi:app --reload
```

### How to Run the Streamlit Frontend

1. Open a new terminal and type the below:

```bash
- Navigate to the backend folder: 'cd streamlit_frontend'
- 'pip install -r requirements.txt'
- Start the API:
'streamlit run AAI4001_FinalProject_Group4_deployment.py'
```

## Why It Matters

Although the dataset is global and not specific to Jamaica, the problem is _highly relevant to the Jamaican agricultural sector_.

Agriculture in Jamaica is vulnerable to climate variability, hurricanes, and droughts, and many farmers have limited access to resources and climate information. This project provides a foundation for future models that can be adapted to local Jamaican conditions to support better planning and decision-making.
